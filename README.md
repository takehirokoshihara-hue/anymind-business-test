# AnyMind ビジネス・プロダクト理解度テスト

社内メンバーがAnyMindのビジネスモデルと業界用語の理解度を確認するためのインタラクティブなWebアプリケーションです。

## 🚀 アクセス

https://takehirokoshihara-hue.github.io/anymind-business-test/

## 📚 テスト内容

- 経営・財務の基本用語
- ビジネス戦略・成長モデル
- SaaS・顧客継続指標
- 広告・マーケティング・SNS
- EC・製造・物流
- 組織・基盤
- プロダクト名
- ビジネスモデル

全36問からランダムに出題されます。

## 🎯 機能

- 4択問題形式
- 即座の正誤判定と解説表示
- プログレスバーによる進捗確認
- 最終スコアと復習リスト
- レスポンシブデザイン（スマホ・タブレット対応）
- 問題と選択肢のランダム化

## 🛠️ 技術スタック

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
